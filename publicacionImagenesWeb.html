Formatos de imagenes en la Web

  • GIF: Graphic Interface Format.
  • JPG/JPGEG: Joint Photograhic Expert Group.
  • PNG: Portable Network Graphics.
  • SVG: Scalable Vector Graphics.

¿Cuáles son sus caracteristicas?

GIF

Este formato tiene un rango de 256 Colores, no siendo el que más calidad ofrece, sin embargo, admite fondo transparente. Otra característica que tiene el formato GIF es la posibilidad de realizar animaciones con programas como Photoshop. Estas animaciones son todavía hoy utilizadas para publicidad o ADS o incluso redes sociales como Instagram.

JPG/JPGEG

Este formato dispone de un rango bastante amplio de color, además permite una buena optimización de peso (lo cual es importante de cara al SEO). Otra de las características importantes es su compatibilidad con la práctica totalidad de dispositivos móviles y navegadores, lo cual le hacen el tipo de formato de imagen más utilizado en la web.

PNG

Tiene unas características muy similares al JPG, aunque permite una mayor profundidad en los contrastes y luces. Es uno de los formatos predeterminados en programas como Word y otros de edición de documentos.

SVG

Este formato es el único formato vectorial de los cuatro listados, cuando hablamos de formato vectorial hablamos de formato escalable y adaptable sin pérdida de calidad (a diferencia de los formatos basados en bitmap como GIF, PNG y JPG). El uso de este formato se ha popularizado en los últimos tiempos en la web debido a la cada vez mayor compatibilidad con dispositivos y navegadores.

WEBP

Estos formatos están pensados para la optimización de las imágenes, no obstante efectividad y compatibilidad no son aún del 100%.

¿Cuál usar?

Usaremos el formato GIF en los siguientes casos
  • Imágenes pequeñas y de poca calidad, por ejemplo, iconos de menú o para las propias redes sociales
  • Animaciones: Como mencionamos anteriormente el formato GIF permite crear animaciones en programas como Photoshop.
  • Fondo transparente: Este formato permite el fondo transparente tanto en imágenes fijas como imágenes animadas.
Cuando usar formato JPG/ JPEG

En general usaremos este formato cuando no necesitemos fondo transparente y la imagen necesite ser optimizada (peso) de cara a la rapidez de carga de la misma. Algunos casos típicos del uso de JPG serían los siguientes:

  • Imágenes de contenido: Ya sea para el blog, páginas web o nuestras redes sociales.
  • Imágenes de producto: Siempre que los mismos no requieran de fondo transparente.
  • Imágenes de fondo: Al permitir una buena optimización su uso es típico en fondos y efectos decorativos.
Cuando usar formato PNG

Usaremos este formato en los casos en que la calidad sea importante y además necesitemos conservar el fondo transparente. Podemos usar el formato PNG en los siguientes casos:

  • Imágenes de contenido (blog, artículos, web).
  • Fotos de producto, si estos llevan fondo transparente.
  • Logos corporativos, ya que el PNG permite más gama de luces y contraste que el JPG
Cuando usar el formato SVG

Usaremos este formato solo si estamos seguros de que nuestros visitantes (por perfil de navegador o dispositivo) utilizan sistemas compatibles con este tipo de archivo. Los casos más utilizados son los siguientes:

  • Iconos
  • Infografías
  • Logo corporativo (ojo con la compatibilidad)

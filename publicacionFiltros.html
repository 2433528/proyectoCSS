Filtros de imagenes en CSS

Los filtros CSS son una característica que permite aplicar ciertos efectos de imagen, propios de aplicaciones de retoque fotográfico, como sepia, variaciones de brillo o contraste, sin hacer cambios permanentes sobre una imagen. Estos funcionan a través de la propiedad filter, a la cuál hay que especificarle una función concreta de las existentes.

Funciones para usar filter

Funcion Descripción Valor
grayscale Escala de blanco y negro porcentaje/numero
blur Desenfoque Gausiano size
sepia Grado de color sepia porcentaje/numero
saturate Grado de saturación porcentaje/numero
opacity Grado de transparencia

La propiedad opacity ya existe en CSS fuera de los filtros CSS como una propiedad independiente. Sin embargo, también existe como función de filtro de CSS.
